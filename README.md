Big Data Handling with Mosquitto, ThingsBoard, and Kafka

This project demonstrates practical experience in big data streaming, IoT data handling, and real-time data engineering using Mosquitto, ThingsBoard, Kafka, and Docker.


Project Overview

In this project, you will learn how to:

- handle large volumes of data using Mosquitto and the MQTT protocol

- set up Mosquitto as a Docker container, and publish and subscribe to messages

- explore data engineering use cases for Mosquitto

- stream big data using ThingsBoard

- configure ThingsBoard in a Docker container and connect it to Firebase

- set up environments, Root Rule Chains, and widgets in ThingsBoard

- pump live data from ThingsBoard to Firebase and apply alarm rules for threshold events

- handle live streaming data at scale using Kafka

- set up Kafka in a Docker container, implement producers and consumers, and build a Node.js web server for real-time data streaming


Key Skills Gained

- implementing MQTT-based data pipelines with Mosquitto

- connecting IoT data streams to ThingsBoard and Firebase

- configuring Rule Chains and alarms in ThingsBoard for real-time monitoring

- building Kafka producers and consumers for large-scale streaming data

- developing web servers with Node.js to process streaming data

- integrating big data tools and real-time analytics workflows

- applying data engineering practices in hands-on projects suitable for a portfolio


Module 24 Projects

- Project 24.1 (Part 1): Set up Mosquitto, ThingsBoard, and Firebase environments; produce and publish temperature and humidity data via MQTT to ThingsBoard and Firebase (~2.5 hours).

- Project 24.2 (Part 2): Configure alarm rule chains in ThingsBoard to send threshold-based alerts to Firebase; connect Root Rule Chain to alarm chain (~2 hours).

- Project 24.3 (Project 2): Implement Kafka to stream vehicle location data; create Python publisher and Node.js consumer; build a real-time web server for data visualization.


Summary

This module provides hands-on experience in IoT data streaming, big data pipelines, and real-time data engineering.
It demonstrates the ability to integrate Mosquitto, ThingsBoard, and Kafka to create fully functional real-time data workflows, making it a portfolio-ready project.
